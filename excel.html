¿Qué es y cómo funciona?

Microsoft Excel es un programa de hoja de cálculo que permite organizar, analizar y representar datos mediante tablas, fórmulas y gráficos. Funciona con celdas organizadas en filas y columnas donde se ingresan datos y se aplican fórmulas para obtener resultados automáticos.

Tabla Dinámica

Permite resumir la información registrada en la base de datos y generar reportes como el total de asistencias o faltas por grupo.

Gráfico Dinámico

Representa visualmente los resultados obtenidos en la tabla dinámica, facilitando el análisis de la información.

Macro

Automatiza tareas repetitivas como registrar asistencia o actualizar reportes. Para guardar una macro, sigue estos pasos: 1 Activa Programador Archivo → Opciones → Personalizar cinta → Marca Programador. 2 Graba la macro Programador → Grabar macro → Ponle nombre → Aceptar. 3 Haz lo que quieres automatizar (Formato, fórmulas, colores, etc.) 4 Detén la grabación Programador → Detener grabación. 5 Ejecuta la macro Programador → Macros → Selecciona → Ejecutar. 6 Guarda el archivo correctamente Guardar como → Libro habilitado para macros (*.xlsm).
